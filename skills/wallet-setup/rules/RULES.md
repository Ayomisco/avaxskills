# Wallet Setup Rules

## Must-Follow
- Always use Core Wallet for the best Avalanche-native experience: `https://core.app`
- Always configure Fuji testnet before mainnet — practice network switching
- Always store seed phrases in a hardware wallet or offline — never in cloud storage or screenshots

## Never Do
- Never share your seed phrase or private key with anyone or any site
- Never use a hot wallet with significant AVAX for development testing
- Never reuse a development wallet for mainnet production funds

## Always Check
- Confirm network is Fuji (chain ID 43113) before testing with testnet AVAX
- Verify wallet shows correct AVAX balance on the correct network before transacting
- Keep a funded backup wallet in case the primary wallet has issues during deployment
