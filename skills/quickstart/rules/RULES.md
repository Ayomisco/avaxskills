# Quickstart Rules

## Must-Follow
- Always use Fuji testnet for first deployments — chain ID 43113, RPC `https://api.avax-test.network/ext/bc/C/rpc`
- Always get testnet AVAX from `https://faucet.avax.network` before proceeding
- Use Core Wallet for the best Avalanche-native experience

## Never Do
- Never use mainnet for learning or testing — Fuji is free and identical
- Never use deprecated Avalanche CLI for new projects
- Never skip the quickstart path — even experienced Ethereum devs need Avalanche network config

## Always Check
- Confirm wallet is connected to Fuji (chain ID 43113) before deploying
- Verify AVAX balance is sufficient for deployment gas (0.1+ AVAX for most contracts)
- Reference `https://docs.avax.network` for latest quickstart guide
